Game Rules

  • You get 1 point for each correct answer without using a hint.
  • If you use a hint, you will earn only 0.5 points for that question.
  • You have 3 attempts to guess each emoji puzzle.
  • Type "hint" to get a hint.
  • Type "pass" to pass the current question.
  • The game consists of 5 rounds. Try to score as high as possible!
